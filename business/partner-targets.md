# partner-targets.md

## Partner Integration Roadmap for Admin Bootcamp

### Target Integrations

1. **Udemy**
   - **Free-Tier Limits**: None; revenue share on course sales.
   - **Integration Effort**: Medium
   - **Value-Add**: Provides access to a vast library of courses on system administration, enhancing the learning path with expert-led content. This integration allows users to purchase and access relevant courses directly from the Admin Bootcamp platform.

2. **GitHub**
   - **Free-Tier Limits**: Unlimited public repositories; 500MB storage for private repositories.
   - **Integration Effort**: Medium
   - **Value-Add**: Facilitates collaboration and version control for projects. Users can manage their code and projects, which is essential for practical learning in system administration.

3. **Slack**
   - **Free-Tier Limits**: 10 integrations; 5GB total file storage.
   - **Integration Effort**: Small
   - **Value-Add**: Enables real-time communication and collaboration among students and instructors. This integration supports community building and peer support, crucial for effective learning.

4. **Trello**
   - **Free-Tier Limits**: Unlimited boards, lists, and cards; 10MB per file attachment.
   - **Integration Effort**: Small
   - **Value-Add**: Provides a visual project management tool for students to organize their learning tasks and track progress. This helps in maintaining motivation and accountability.

5. **AWS Educate**
   - **Free-Tier Limits**: $100 in AWS credits for students.
   - **Integration Effort**: Medium
   - **Value-Add**: Offers hands-on experience with cloud computing, which is essential for modern system administration. This integration can provide students with practical labs and resources.

6. **LinkedIn Learning**
   - **Free-Tier Limits**: 1-month free trial; subscription-based thereafter.
   - **Integration Effort**: Medium
   - **Value-Add**: Access to a library of professional courses on system administration and related fields. This integration can enhance the curriculum with additional resources and certification opportunities.

7. **Zapier**
   - **Free-Tier Limits**: 5 Zaps; 100 tasks per month.
   - **Integration Effort**: Medium
   - **Value-Add**: Automates workflows between Admin Bootcamp and other applications, streamlining user experience and administrative tasks. This can enhance user engagement and retention.

8. **Discord**
   - **Free-Tier Limits**: Unlimited members and channels; 8MB file upload limit.
   - **Integration Effort**: Small
   - **Value-Add**: Provides a platform for community engagement and support. Users can join study groups, share resources, and collaborate on projects, fostering a supportive learning environment.

### Summary
The integrations listed above are prioritized based on their potential to enhance the user experience, provide valuable resources, and create opportunities for revenue-sharing partnerships. Each integration is designed to address specific user jobs within the learning journey of system administration, ensuring that Admin Bootcamp remains a comprehensive and effective platform for students.
